## Plan de 2 días — próxima sesión (5-6 abril 2026)

### Contexto
- **500+ libros** listos para cargar en la primera etapa
- **150+ vendedores** históricos de tuslibros.cl para re-contactar
- MercadoPago Split Payment es la pieza crítica que faltaba

### Día 1 (5 abril): MercadoPago Split Payment + Navegación

**Mañana — MP Split Payment (prioridad máxima)**
1. Implementar OAuth de MercadoPago para vendedores
   - Flujo: vendedor conecta su cuenta MP desde `/perfil`
   - Guardar `mp_access_token` y `mp_user_id` en tabla `users` (Supabase)
2. Crear endpoint `/api/mp/connect` para callback OAuth
3. Modificar checkout para usar split payment:
   - Crear preferencia con `marketplace_fee` (comisión tuslibros)
   - Dinero va directo al vendedor, fee se retiene
4. Webhook `/api/mp/webhook` para confirmar pagos y actualizar estado de orden

**Tarde — Navegación y UX**
5. Mover Navbar a layout compartido (eliminar 19 imports duplicados)
6. Sidebar de categorías en mobile (drawer/bottom sheet)
7. Mejorar experiencia de búsqueda (autocompletado)

### Día 2 (6 abril): Arriendos + Pulido final

**Mañana — Sistema de arriendos**
1. Flujo completo de arriendo: seleccionar período (7/14/30 días), garantía, checkout
2. Notificaciones de devolución (email o in-app)
3. Estado de arriendo en `/mis-arriendos`

**Tarde — Preparación para lanzamiento**
4. Carga masiva de los 500+ libros (script o admin panel)
5. Testing end-to-end del flujo completo: publicar → buscar → comprar/arrendar → pagar → despacho
