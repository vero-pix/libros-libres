# Prompt para IA — Carrusel LinkedIn tuslibros.cl

## Contexto (copia-pega a ChatGPT primero)

Soy Verónica Velásquez, fundadora de tuslibros.cl, marketplace de libros usados chileno.

Necesito un **carrusel de LinkedIn de 5 slides** (formato cuadrado 1080×1080 px) para publicar mañana, Día del Libro. Estilo visual: tipografía serif editorial para títulos (estilo Playfair Display o Canela), sans-serif para datos (Inter/Helvetica). Paleta: fondo cream/beige cálido (#faf7f2 / #fef9ed), acentos en ámbar/dorado (#d4a017), texto en ink oscuro NO puro (#2a1f14). Nada plano. Permitido: texturas de papel sutiles, sombras suaves, subrayados manuscritos. Prohibido: stock photos genéricos, emojis de Windows, gradientes ochentosos.

## Si la IA no puede generar el carrusel visual directo

Pídele que te dé:
1. Una **descripción DALL-E** para cada slide (formato: "Create an image 1080x1080 px, background #faf7f2, serif title…")
2. O que te exporte el contenido en un **formato importable a Canva** (markdown estructurado + descripción visual)
3. Colores en HEX ya listos para que los copies
