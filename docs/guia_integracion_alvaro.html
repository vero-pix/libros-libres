Conecta tu tienda PrestaShop
con tuslibros.cl

Guía técnica paso a paso · Mayo 2026 · Preparada para Librería Comuna Literaria
¿Para qué sirve esto? Con esta integración, tu catálogo de PrestaShop se publica automáticamente en tuslibros.cl. Cuando se agote stock en tu tienda, el listado se pausa solo. Sin trabajo manual, sin duplicar datos.

Requisitos previos

Del lado de PrestaShop

  • PrestaShop 1.7 o superior
  • Webservice activado (Parámetros Avanzados)
  • API Key con permisos de lectura
  • Node.js 18+ instalado en tu computador

Del lado de tuslibros

  • Cuenta registrada en tuslibros.cl
  • Perfil completado (tienda + MercadoPago)
  • API Key generada en /perfil

Paso a paso

1

Activa el Webservice de PrestaShop

En el back-office de tu tienda:

  • Ve a Parámetros Avanzados → Webservice
  • Activa el webservice (interruptor en ON)
  • Haz clic en "Agregar nueva clave de webservice"
  • En permisos, marca GET para: products, stock_availables
  • Guarda y copia la clave generada (44 caracteres)
Ejemplo de clave: ZDZFH8XJK3BYWPQ2MNVRLTE9IA0GSOC6 — es solo lectura, no permite modificar nada en tu tienda.
2

Genera tu API Key en tuslibros.cl

Inicia sesión en tuslibros.cl y ve a /perfil → API (o pídele a Veronica que te genere una). Se ve así:

tl_a3f9b2c1e8d7...

Esta clave identifica tu tienda al crear y actualizar listados.

3

Configura el archivo de variables

Descarga los scripts y crea un archivo .env.local en la misma carpeta con estos valores:

# URL base de tu tienda PrestaShop (sin slash final) PS_URL=https://comunaliteraria.cl # API Key del webservice de PrestaShop PS_API_KEY=ZDZFH8XJK3BYWPQ2MNVRLTE9IA0GSOC6 # API Key de tuslibros (desde tu perfil) TL_API_KEY=tl_a3f9b2c1e8d7... # (Opcional) Probar sin publicar nada DRY_RUN=true
4

Importa tu catálogo (una sola vez)

Este script lee todos tus productos en PrestaShop y crea los listados en tuslibros. Solo importa productos con stock disponible.

Primero, prueba en modo simulación:

DRY_RUN=true node scripts/import_prestashop.mjs

Cuando el resultado se vea bien, ejecuta la importación real:

DRY_RUN=false node scripts/import_prestashop.mjs

Verás un progreso en pantalla:

═══════════════════════════════════════════════════ IMPORTADOR PrestaShop → tuslibros.cl Tienda: https://comunaliteraria.cl ═══════════════════════════════════════════════════ 📦 Obteniendo lista de productos... → 1.247 productos encontrados en total → Procesando 1.247 productos [1247/1247] Procesando... (creados: 891, saltados: 312, errores: 44) ✅ Importación completada: Creados: 891 Saltados: 312 (sin stock o datos incompletos) Errores: 44
Tip: Para importar solo una parte y probar, agrega MAX_BOOKS=50 al comando.
5

Activa la sincronización automática de stock recomendado

Este script compara el stock de PrestaShop con el estado de tus listados en tuslibros. Si un libro se agota en tu tienda, lo pausa en tuslibros. Si vuelve a haber stock, lo reactiva.

node scripts/sync_prestashop.mjs

Para que se ejecute automáticamente cada 30 minutos, abre el crontab de tu computador/servidor:

crontab -e

Y agrega esta línea (ajusta la ruta según dónde dejaste los scripts):

*/30 * * * * cd /ruta/a/los/scripts && node sync_prestashop.mjs >> sync.log 2>&1

El log de cada ejecución se verá así:

[08/05/2026, 14:30:00] Iniciando sync PrestaShop ↔ tuslibros... tuslibros: 891 activos, 0 pausados PrestaShop: 1.247 productos con stock info ⏸ Pausando: "Historia de Chile Vol. 1" (stock PS: 0) ⏸ Pausando: "El Mercurio de Chile" (stock PS: 0) Sync completado: Pausados: 2 Reactivados: 0 Sin cambio: 889 No encontrados en PS: 0

Variables de configuración

Variable Descripción
PS_URL URL base de tu tienda PrestaShop
PS_API_KEY Clave del webservice de PrestaShop
TL_API_KEY Tu API Key de tuslibros.cl
TL_URL URL de tuslibros (default: https://tuslibros.cl)
DRY_RUN "true" = simular sin publicar nada
MAX_BOOKS Límite de libros a importar (solo para pruebas)

Preguntas frecuentes

¿Qué pasa si un libro ya existe en tuslibros?
El importador crea un nuevo listado. No actualiza ni duplica el existente. Si quieres evitar duplicados, ejecuta la importación solo una vez o contacta a Veronica para revisar antes.
¿Puedo filtrar por categoría de PrestaShop?
Sí. Próximamente se agrega la variable PS_CATEGORY_ID para importar solo libros de una categoría específica.
¿Cuándo se refleja el cambio de stock en tuslibros?
Depende de con qué frecuencia ejecutes el script de sync. Con la configuración recomendada (cada 30 min), el desfase máximo es 30 minutos.
¿El importador sube las fotos de los libros?
En la versión actual no. tuslibros busca la portada automáticamente por ISBN (desde Open Library y otras fuentes). Si el libro tiene ISBN, la foto aparece sola. Para libros sin ISBN, puedes subir la foto manualmente desde el panel.
¿Qué comisión cobra tuslibros?
8% del precio de venta, solo cuando se concreta la transacción. Sin cuota mensual ni costo de publicación.
¿Necesito saber programar para usar esto?
No, pero necesitas acceso al back-office de PrestaShop para activar el webservice, y tener Node.js instalado en tu computador para ejecutar los scripts. Si necesitas ayuda, Veronica puede asistirte en la primera configuración.
